# microservices-event-bus

> Event-driven microservices communication layer with RabbitMQ and Kafka adapters

## Getting Started

```bash
npm install
npm start
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Start the application |
| `npm test` | Run test suite |
| `npm run lint` | Run ESLint |
